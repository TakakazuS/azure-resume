Azure Resume

Website

  • The project is a fully Azure-hosted resume consisting of a frontend and backend. The frontend is a static website, while the backend is an Azure Function integrated with Cosmos DB. The backend retrieves and increments visitor counter data stored in Cosmos DB. Deployment involves publishing the Azure Function, creating a storage account, and deploying the frontend to Azure Storage. A CI/CD pipeline automates code deployment, and unit testing is implemented. The project showcases a concise and scalable Azure-based solution for hosting a resume with efficient development and deployment processes.

    • Tollbooth App

    Github repository

  • The Tollbooth App is a serverless application hosted in Azure. It utilizes Azure Functions, Cognitive Services, Event Grid, Application Insights, Cosmos DB, and Logic Apps. The app processes vehicle photos, extracting license plate data using OCR and storing it in Cosmos DB. Uploaded photos trigger an Azure Function, successful results are stored in Cosmos DB, while unsuccessful ones prompt manual checkups. A scheduled Logic App exports data from Cosmos DB to Blob storage. The Tollbooth App demonstrates seamless integration and scalability in Azure for efficient license plate data processing and storage.
